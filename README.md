# Speed Test

Same-origin speed test for Cloudflare.

The page measures download speed, upload speed, idle latency, jitter, and loaded
latency against the domain it is served from. There is no domain-specific config
in the app, so the same build can be deployed to multiple hostnames. Transfer
payloads are generated test bytes.

## Run locally

```sh
npm run dev:local
```

Open:

```txt
http://127.0.0.1:4173
```

## Test

```sh
npm test
```

## Cloudflare

Static files live in `public/`.

The Worker entrypoint is `src/worker.js`. API handlers are shared from
`src/api.js`.

- `/api/download` returns generated bytes for download tests.
- `/api/upload` accepts generated upload bytes and discards them.
- `/api/trace` returns basic request metadata, if needed later.

Worker static-assets deployments use `wrangler.jsonc` and `src/worker.js`.
The `functions/` directory contains Pages Function wrappers for compatibility
with Cloudflare Pages.

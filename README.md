# Speed Test

Same-origin speed test for Cloudflare Pages.

The page measures download speed, upload speed, idle latency, jitter, and loaded
latency against the domain it is served from. There is no domain-specific config
in the app, so the same build can be deployed to multiple hostnames. Transfer
payloads are generated test bytes.

## Run locally

```sh
npm run dev:local
```

Open:

```txt
http://127.0.0.1:4173
```

## Test

```sh
npm test
```

## Cloudflare Pages

Static files live in `public/`.

Pages Functions live in `functions/`:

- `functions/api/download.js` returns generated bytes for download tests.
- `functions/api/upload.js` accepts generated upload bytes and discards them.
- `functions/api/trace.js` returns basic request metadata, if needed later.
